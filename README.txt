Getting Started 
Scan the QR code of the publication in the expo at the link:
https://expo.io/@bohdanshevchenko/projects/calculator

or running locally:

1. Clone repository https://github.com/bohdanshevchenkoA/Codempire_calculator.git
2. npm i
3. npm start
4. Scan the QR code through the Expo mobile app

To run the tests: npm run test
